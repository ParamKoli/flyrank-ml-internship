Content Refresh Opportunity Scoring using Machine Learning

FlyRank ML Internship Capstone

Author: Param Koli

Abstract

This capstone project predicts whether existing web content should be refreshed based on historical performance data. The objective is to help prioritize content updates using machine learning instead of manual review. A Random Forest classifier was trained and evaluated against a baseline model to identify pages likely to experience downward performance trends.

Introduction

Content refresh is an important SEO strategy. Instead of updating every page, organizations benefit from identifying pages that are most likely to lose traffic. This project applies supervised machine learning to classify content into refresh candidates based on historical metrics.

Dataset

Dataset Used:

  • content_refresh_anonymized.csv
  • 30,000 rows
  • 44 columns
  • Trend labels were excluded from model features to avoid data leakage.

Methodology

  1. Load dataset.
  2. Clean missing values.
  3. Separate target variable.
  4. Train/Test split.
  5. Create baseline model.
  6. Train Random Forest classifier.
  7. Evaluate using Accuracy, Precision, Recall and F1 Score.

Results

Metric Value
Baseline Accuracy 0.700
Random Forest Accuracy 0.689
Precision 0.700
Recall 0.752
F1 Score 0.725

Discussion

Although the Random Forest model achieved slightly lower accuracy than the baseline, it significantly improved recall. This means the model successfully identifies more pages that require refreshing, making it more useful in practical SEO workflows.

Recommendations

  • Use additional behavioral features.
  • Perform feature engineering.
  • Tune Random Forest hyperparameters.
  • Evaluate Gradient Boosting and XGBoost.
  • Retrain the model periodically using fresh data.

Limitations

  • Single anonymized dataset.
  • No hyperparameter optimization.
  • Limited feature engineering.
  • No external SEO signals.

Conclusion

The project demonstrates that machine learning can assist in identifying web pages that should be refreshed. Random Forest produced strong recall and F1 score, making it valuable for prioritizing content updates. Future improvements include richer features and advanced ensemble models.
